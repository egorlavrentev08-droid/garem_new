# 🦊 Dori — Архитектор Дисциплины

**Dori** — это Telegram-бот для развития дисциплины и ежедневных привычек. Бот отслеживает активность пользователей в чате, начисляет монеты за сообщения, выдаёт ранги за длинные стрики и напоминает о себе, если вы пропали.

## 🎯 Основные возможности

| Механика | Описание |
|----------|----------|
| 📊 **Стрик** | Количество дней подряд, в которые вы были активны в чате |
| 🪙 **Монеты** | Начисляются за каждое сообщение в основном чате |
| 🏆 **Ранги** | 9 рангов, повышаются в зависимости от длины стрика |
| 🛡️ **Щит** | Защищает стрик от сброса на 36 часов |
| 🔄 **Искупление** | Шанс восстановить сброшенный стрик |
| 🚫 **Бан** | Автоматический бан через месяц неактива |
| 🏅 **Топы** | Ежедневные и еженедельные награды |
| 🦊 **Дори** | Живой персонаж с сотнями уникальных фраз |

## 🔧 Как это работает

### Стрик
- Стрик растёт **только если вы написали 100+ сообщений за день** в основном чате
- Максимум **+1 стрик в день**
- Стрик сбрасывается, если вы не писали **24 часа**

### Монеты
- За каждое сообщение в основном чате вы получаете монеты
- Доход зависит от вашего ранга (от 0.1 до 0.5 монеты за сообщение)

### Ранги
| Ранг | Минимальный стрик | Доход за сообщение |
|------|-------------------|-------------------|
| Без ранга | 0 | 0.10 |
| Новичок | 1 | 0.10 |
| Кандидат | 2 | 0.15 |
| Знакомый | 5 | 0.20 |
| Хороший | 9 | 0.25 |
| Душа | 16 | 0.30 |
| Старожил | 24 | 0.35 |
| Гордость | 34 | 0.40 |
| Авторитет | 47 | 0.45 |
| Незаменимый | 60 | 0.50 |

## 📖 Команды

### Пользовательские команды

| Команда | Описание |
|---------|----------|
| `/start` | Зарегистрироваться и получить приветствие |
| `/me` | Показать профиль (стрик, монеты, ранг) |
| `/name` | Сменить имя в боте |
| `/shop` | Магазин (купить или активировать щит) |
| `/top` | Топы (стрики / сообщения за сегодня) |
| `/redemption` | Статус восстановления стрика |

## 🛡️ Система щитов

- **Цена:** 100 монет
- **Эффект:** защищает стрик от сброса на 36 часов
- **Покупка:** `/shop` → кнопка "Купить щит"
- **Активация:** `/shop` → кнопка "Активировать щит"
- **Хранение:** щиты хранятся в запасе, можно купить несколько

---

## 🔄 Искупление

Если стрик был сброшен:
1. У вас есть **24 часа**, чтобы восстановить его
2. Нужно написать **200 сообщений** в основном чате
3. Прогресс можно отслеживать через `/redemption`
4. Если не успели — стрик теряется навсегда

---

## 🚫 Бан за неактив

- Если пользователь не писал **30 дней** — он автоматически блокируется
- Забаненный пользователь не получает монеты, не участвует в топах
- Бот полностью игнорирует сообщения от забаненных пользователей
- Админы могут разбанить через `/dpardon`

## 🦊 Фразы и картинки

Дори — это не просто бот. У него есть характер.

### Триггеры фраз

| Триггер | Описание | Настроение |
|---------|----------|------------|
| `DAILY_GREETING` | Ежедневное приветствие | Саркастичное |
| `ONE_DAY_INACTIVE` | 1 день неактива | Злое |
| `MULTI_DAY_INACTIVE` | Несколько дней неактива | Гневное |
| `MULTI_DAY_ULTRA` | Вечность неактива | Гневное |
| `RETURN` | Возвращение после пропуска | Саркастичное |
| `RETURN_AGO` | Возвращение после долгого пропуска | Саркастичное |
| `RANK_UP` | Повышение ранга | Саркастичное |
| `STREAK_ACHIEVEMENT` | Достижение стрика (7,14,31,99,356) | Саркастичное |
